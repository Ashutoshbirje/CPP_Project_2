# HELP GUIDE

  How to Use the Hotel Management System

------------------------------------------------------------------------------

1. Home Page

  The Home Page provides navigation to different sections of the system:

  About: View detailed information about the hotel.

  Customer: Access customer-related features.

  Help: Get assistance on using the software.

  Settings: Modify software settings (requires a password).

------------------------------------------------------------------------------

2. Customer Section

   Click on Customer to access three options:

   Menu Card: 
    - View the available menu items and prices. 
    - Select items, and the bill will be generated automatically.

   Back to Home: 
    - Return to the home page.

------------------------------------------------------------------------------

3. Help Section

    Click on Help to view instructions on using the software, FAQs.

------------------------------------------------------------------------------

4. Settings

    The Settings section allows changes to software configurations. 
    A password is required to modify settings to ensure security.
